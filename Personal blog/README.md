# personalBlogWebsite
Personal Blog Website using HTML5, CSS3 Grid and Flexbox

## Tools used

- Webfonts: [Google Webfonts Helper](https://google-webfonts-helper.herokuapp.com/fonts)

## Where did I use Flexbox?

- Header
- Section
- Article - card
- Footer

## Where did I use CSS Grid?

- Body
- Section - Blog
- Section - Blog - single
